Module naive 

Source
Expand description

Naive bidirectional implementation (the block itself is unchanged; a forward and a reversed pass are merged). Naive bidirectional Mamba-3: the block itself is not adapted.

Each Mamba3BidiLayerPair runs two independent layers — a straight (→) pass and a reversed (←) pass over the sequence axis. The reversal happens before the layer normalization, and the two directions are merged (by an [OutputMerge]) after the block output, before the layer-pair skip connection.
