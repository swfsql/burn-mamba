Expand description

Apply rotary position embeddings to x along its last dimension.

Two pairing conventions are supported, selected by rotate_pairwise:

  • rotate_pairwise = true — interleaved (NeoX / Triton style): adjacent pairs (0,1), (2,3), … are rotated together. Used by the SISO Triton kernel (mamba3_siso_*.py).
  • rotate_pairwise = false — half-and-half (GPT-J style): position n is paired with n + N/2. Used by the MIMO Tilelang kernel (mamba3_mimo_fwd.py).

Reference: mamba3.py:335 sets rotate_pairwise = not self.is_mimo.

§Shapes

  • x: [..., state_rank] where state_rank is even
  • angles: [..., state_rank / 2] (one angle per pair)
  • output: same shape as x
