Expand description

Hyperparameters for the Mamba-2 SSM block.

All computed quantities (e.g. nheads, d_inner, conv_dim) are derived from the stored fields; see the helper methods on Mamba2Config.

Fields§

§d_model: usize

Model (hidden) dimension D. Every token is represented as a D-dimensional vector at the input and output of the block.

§state_rank: usize

State rank N — the latent dimension of the SSM hidden state.

Larger N gives a more expressive state but increases memory and compute per step. The paper uses N ∈ {64, 128, 256} for most experiments.

§conv_kernel: usize

Causal convolution window length. Typically 4.

§expand: usize

Expansion factor for d_inner = expand · d_model.

An expansion of 2 doubles the internal width, keeping the parameter count of the SSM block comparable to a standard attention layer.

§per_head_dim: usize

Head dimension P. The total d_inner is split into nheads = d_inner / P independent SSM heads.

Typical values: 64 or 128. Smaller P means more heads and a larger hidden state per model dimension.

§ngroups: usize

Number of B/C groups G. Must divide nheads.

Setting G < nheads reduces the B and C projection sizes (analogous to GQA in attention), saving memory without a large accuracy cost.

§a_init_range: (f64, f64)

Range [lo, hi] for the uniform initialisation of the magnitude of A.

A = -Uniform(lo, hi), stored as a_log = log(Uniform(lo, hi)). The paper uses [1, 16] by default.

§is_norm_before_gate: bool

Gated RMSNorm mode: when true the norm is applied before the gate; when false (default) the gate is applied first (SiLU-gated norm).

§dt_min: f64

Minimum value of the initial Δ distribution. Used to set dt_bias.

§dt_max: f64

Maximum value of the initial Δ distribution. Used to set dt_bias.

§dt_init_floor: f64

Floor clamped onto the sampled initial Δ values before inverting to obtain dt_bias. Prevents numerical issues with very small Δ.

§dt_limit: (f64, f64)

Hard clamp limits for Δ at runtime: Δ ∈ [dt_limit.0, dt_limit.1].

Defaults to (0, f16::MAX ≈ 65504), effectively only clamping at 0.

§has_proj_bias: bool

Whether to add a bias term to the in_proj and out_proj projections.

§has_conv_bias: bool

Whether to add a bias term to the depthwise convolution.

§has_learnable_init_states: bool

Whether to allocate a learnable initial SSM state h₀.

When false (default), the hidden state starts at zero for every sequence. When true, init_states_hpr is allocated as a trainable parameter of shape [nheads, per_head_dim, state_rank].

pub fn nheads(&self) -> usize

nheads H = d_inner / per_head_dim.

Source

pub fn conv_dim(&self) -> usize

conv_dim = d_inner + 2 · ngroups · state_rank.

The depthwise convolution processes x, B, and C concatenated: x contributes d_inner channels, B and C each contribute ngroups · state_rank channels.
