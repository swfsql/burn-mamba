Macros emitting per-backend BackendExt impls + autodiff marker traits. Macros to cut down the per-backend boilerplate for *SsdBackendExt traits.

Both Mamba-2 and Mamba-3 define an SSD-backend extension trait whose default implementation already does the right thing for every burn backend except Autodiff (which gets the custom backward). The macros below emit the per-backend “use the default impl” blocks and the autodiff marker trait.
