§Mamba-1 Language Model Network

Assembles a complete autoregressive language model from the Mamba-1 components:

  tokens [batch, sequence]
      │  Embedding (vocab_size → d_model)
      ▼  (×n_virtual_layers)
  Mamba1Layer [Pre-LN residual block]
      │  RMSNorm (final)
      ▼  LM head (d_model → vocab_size)
  logits [batch, sequence, vocab_size]

Mirrors crate::mamba2::network: vocabulary is padded up to pad_vocab_size_multiple, the LM head can be weight-tied to the embedding (missing_lm_head = true), and both Mamba1Network::forward and Mamba1Network::step thread a Mamba1Caches so prefill can be followed by decoding.
